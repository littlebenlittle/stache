
# Stache

Stache is an extensible mustache-style templating engine.

## Basic Use

Raku script:

```raku
{{ basic-example }}
```

Output:

```raku
{{ basic-example-output }}
```

## Structure Blocks

### With Blocks

Raku script:

```raku
{{ with-example }}
```

Output:

```raku
{{ with-example-output }}
```

### For Blocks

Raku script:

```raku
{{ for-example }}
```

Output:

```txt
{{ for-example-output }}
```
